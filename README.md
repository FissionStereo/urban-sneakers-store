# Urban Sneakers Store

Tienda online moderna tipo e-commerce desarrollada con HTML, CSS y JavaScript puro.

## 🚀 Demo

🔗 Ver proyecto online:  
https://TUUSUARIO.github.io/urban-sneakers-store/

## 🛠 Tecnologías

- HTML5
- CSS3
- JavaScript
- Fetch API
- JSON
- localStorage
- Git y GitHub
- GitHub Pages

## ✨ Funcionalidades

- Catálogo dinámico de productos
- Tarjetas de productos
- Buscador en tiempo real
- Carrito de compras
- Aumentar y disminuir cantidades
- Eliminar productos
- Cálculo automático del total
- Guardado con localStorage
- Diseño responsive
- Deploy con GitHub Pages

## 📌 Flujo de trabajo Git

El proyecto fue desarrollado usando ramas:

- `main`
- `develop`
- `feature/header-navbar`
- `feature/product-cards`
- `feature/shopping-cart`
- `feature/search-filter`
- `feature/responsive-mobile`
- `feature/github-pages-deploy`
